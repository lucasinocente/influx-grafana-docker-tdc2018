# Node + Influx + Grafana

## Requisitos

- Node
- Docker

## Rodar Influx + Grafana

`docker-compose up`

## Rodar aplicação

```
cd /app
npm install
npm start
```

## Acessar Grafana:

http://localhost:3000

## Configuração Influx:

```
Name: Influx
URL: http://influxdb_1:8086
Access: Browser
Database: tdcpoa2018
```
